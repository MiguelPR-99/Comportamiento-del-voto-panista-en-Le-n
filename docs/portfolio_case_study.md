# Portfolio Case Study - Geospatial Electoral Visualization

## One-line CV description
Built an interactive geospatial editorial map using Next.js, TypeScript, and MapLibre to analyze PAN vote variation between 2018 and 2021 across 846 electoral sections in León, Guanajuato.

## Recruiter-friendly summary
This project converts official electoral and territorial data into a public-facing interactive map designed for clear interpretation, not technical overload. It demonstrates the ability to structure ambiguous data, build a polished digital deliverable, and communicate results in a concise, decision-oriented format.

## Technical summary
The application uses Next.js, React, TypeScript, and MapLibre GL JS to render a bivariate choropleth (3x3 + `no_data`) over 846 sections in León. The workflow includes spatial data preparation, controlled classification logic, explicit missing-data handling (no zero imputation), and reproducible image export through both UI and Playwright CLI.
